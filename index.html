Six layers between your pen and its answer

The network learns edges first, then shapes, then digits — each layer narrows a raw grid of pixels into ten confidence scores.

The model

A compact CNN — two convolution + pooling stages feeding a 128-unit dense layer — trained on handwritten digits. It has 225,034 parameters, all listed below, and was exported directly from the uploaded .h5 file.

conv(32) → pool → conv(64) → pool → dense(128) → dense(10)

How the drawing becomes input

Your stroke is cropped to its bounding box, scaled to fit a 20×20 square, and centered in a 28×28 canvas — the same normalization used when this network was trained, so a small, off-center scribble reads as reliably as a large centered one.
